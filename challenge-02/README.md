# Challenge 02

Develop a CRUD (Create, Read, Update and Delete) of users.

#### Requirements

- Use localStorage to persist data.
- Bootstrap or tailwind.
- VueJS
- Propose the design

**User data structure:**

```json
{
  "id": 1,
  "name": "Jeff",
  "lastname":"musk",
  "age": 24,
  "created_at": "2021-10-22 12:43:02"
}
```
